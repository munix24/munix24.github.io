Lottery Ticket Simulator

This tool allows you to simulate lottery tickets and counts how many matches there are with the winning lottery numbers. Here's how it works:

  1. A set of "winning" numbers is chosen at random automatically
  2. Enter the number of tickets you want to simulate in the input field and click the "Simulate" button to run the simulation for a fixed number of tickets.
    OR Click the "Simulate Until Win" button to continuously simulate tickets until a ticket with all 6 matches is created.
  3. The table below will display the counts of tickets and the number of matches with the winning lottery numbers.
  4. You can click the "Reset" button to reset all tickets, generate new winning numbers, and start fresh.

Each ticket contains 6 numbers, each ranging from 1 to 49. A winning ticket is one that matches all 6 winning numbers the probability of which is approximately 1 in 13,983,816.
